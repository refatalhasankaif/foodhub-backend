# FoodHub Backend API

A robust REST API for the **FoodHub** meal ordering platform.  
Provides complete backend functionality for customers to browse and order meals, providers to manage menus and orders, and admins to oversee the platform.

## Live API
**Backend URL:** https://foodhub-backend-3poi.onrender.com/

## Built With
- **Node.js**
- **TypeScript**
- **Express.js**
- **PostgreSQL**
- **Prisma ORM**
- **Neon DB**
- **Better Auth**
