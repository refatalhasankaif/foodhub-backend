# FoodHub Backend

A robust REST API that powers the FoodHub meal ordering platform. It manages users, meals, orders, and role-based workflows with a strong focus on performance, security, and scalability.

## Tech Stack

- Node.js
- Express.js
- TypeScript
- PostgreSQL (Neon DB)
- Prisma ORM
- Better Auth

---

## Features

- Role-based authentication (Customer, Seller, Admin).
- Secure password hashing using bcrypt.
- Full CRUD for meals, users, and orders.
- Order management system with status tracking.
- Seller APIs for menu and order handling.
- Admin APIs for full platform control.
- Scalable and modular architecture.
- Production-ready build system.
